# Intro
- Spectrum은 다양한 주제와 다양한 시각으로 다가가는 블로그 서비스입니다.
- IT 기술, 문화, 취미 등 다양한 주제를 다루며, 저의 생각과 경험을 바탕으로 각종 회고와 이야기를 공유합니다.
- 함께 성장하는 즐거움을 느낄 수 있는 Spectrum에서 다양한 소통의 창구가 되어보세요.
